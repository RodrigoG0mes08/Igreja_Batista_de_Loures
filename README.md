# Igreja_Batista_de_Loures
Repositório para guardamos o nosso projeto da igreja batista de loures


Na app ter uma aba de eventos, onde apenas os admins podes adicionar eventos e os menbros podem se candidatar a esses eventos, o admin tem que colocar os dados do evento, como por ex, a hr, a idade necessária, o preço (caso seja preciso), etc. 

A conta vai poder associar filhos se quiser, ou seja um pai vai associar um filho á sua conta, para quando houver um evento para pessoas menores de idade os pais possam candidatar os filhos. 

Se for necessário realizar um pagamento, dar para pagar direto para a conta da igreja.
